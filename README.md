 Customer Segmentation using K-Means Clustering

 Overview

This project focuses on segmenting customers based on their demographic and spending behaviour using the K-Means clustering algorithm. The goal is to identify meaningful customer groups that can help businesses improve marketing strategies and decision-making.



Dataset

The dataset contains the following features:

* Customer ID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1–100)

For clustering, the most relevant features used are:

* Annual Income (k$)**
* Spending Score (1–100)**


 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Data visualization using plots and charts
* Feature scaling using StandardScaler
* Optimal cluster selection using the Elbow Method
* Customer segmentation using K-Means clustering

 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn


 Exploratory Data Analysis

* Analyzed gender and age distribution
* Visualized income vs spending behaviour
* Identified patterns and trends in customer data


 Methodology

1. Data cleaning and preprocessing
2. Feature selection and scaling
3. Applying the Elbow Method to determine optimal clusters
4. Training K-Means clustering model
5. Visualizing customer segments


 Results

The model successfully segmented customers into different groups such as:

* High Income – High Spending
* High Income – Low Spending
* Low Income – High Spending
* Low Income – Low Spending
* Average Customers

These insights can be used for targeted marketing and business growth.


 Conclusion

This project demonstrates how unsupervised machine learning techniques like K-Means clustering can be used to extract valuable insights from customer data. The segmentation helps businesses understand customer behaviour and design effective marketing strategies.
